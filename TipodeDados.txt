Tipo de dados em Javascript

Podemos verificar o tipo de dado através da função typeof


Number - Tipo de dado para números, não existe
um tipo definido para inteiros, portantos todos os números são considerados do tipo Number

String - Tipo de dado para textos, as strings podem ser escritas com aspas simples
ou duplas; um número entre aspas pode ser considerado como uma string; é possível concatenar strings com +


Boolean - Representa um valor verdadeiro ou falso; Comparações resultam em boolean;
podemos atribuir um 'true' ou 'false' para uma variável e ela terá o tipo de Boolean 

Null and Undefined - Também são considerados tipos de dados, null é um tipo de dado que representa um valor e Undefined
é um tipo de dado para uma variável com valor não atribuído.

Object - Funcionam como um array associativo de outras linguagens, podemos criar propriedades com chave e valor. A 
idéia é guardar um conjuto de valores para utilizar posteriormente.
